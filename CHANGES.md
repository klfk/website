# CHANGES

## /school section — IDAF-Projekt «Erfolgversprechende Bewerbung» (2026-06-10)

Added a complete `/school/*` sub-site to `ivanm.xyz` as an online application
profile for an IMS Basel school project. Existing routes and the global layout
are untouched.

---

### New files

#### Routes (`src/routes/school/`)

| File | Purpose |
|---|---|
| `+layout.svelte` | Shared layout: sticky 5-item nav, dark/light toggle, search trigger, footer. Scoped CSS custom properties for both themes. Overrides the VT323 body font with a system sans-serif for readability. |
| `+page.svelte` | **Home** — hero with avatar placeholder, Kurzprofil (verbatim text), quick-links, contact form (mailto fallback + honeypot spam guard). |
| `about/+page.svelte` | **Über mich** — full HTML CV: Personalien (dl), Schulbildung, Berufserfahrung, Sprachkenntnisse, Informatikkenntnisse, Projekte, Freizeit, Referenzen. |
| `projects/+page.svelte` | **Projekte** — 4 ProjectCard components + "Mehr auf GitHub" link. |
| `skills/+page.svelte` | **Kenntnisse** — 7 skill groups rendered with SkillBar components, IDAF 1–5 scale, legend at bottom. |
| `documents/+page.svelte` | **Dokumente** — confidentiality note + DocumentItem list. `noindex` meta tag. Protected by Cloudflare Access (see README). |
| `README.md` | Placeholder list, Cloudflare Access setup steps, how to add/remove documents, how to swap company name. |

#### Components (`src/lib/components/school/`)

| File | Purpose |
|---|---|
| `SkillBar.svelte` | Horizontal progress bar for a skill (name, level 1–5). |
| `ProjectCard.svelte` | Card showing title, description, tech tags, link, and "Gelernt" one-liner. |
| `DocumentItem.svelte` | List item with filename, description, file size, download button (disabled when `available: false`). |
| `SchoolSearch.svelte` | Full-screen search modal. Static 30-item index covering pages, projects, skills, and CV facts. Keyboard shortcut ⌘K / Ctrl+K. |

#### Static assets (`static/school/documents/`)

8 placeholder PDFs generated by `scripts/gen-placeholder-pdfs.mjs`.
Replace each with the real document before going live.

#### Scripts (`scripts/`)

| File | Purpose |
|---|---|
| `gen-placeholder-pdfs.mjs` | Generates minimal valid PDF placeholders. Run with `node scripts/gen-placeholder-pdfs.mjs`. |

### Bonus features implemented

- **Dark/Light toggle** — reads `prefers-color-scheme` on first visit, persists to `localStorage`. Toggle button in the nav.
- **Search** — client-side, ⌘K shortcut, 30-item static index.
- **Contact form** — `mailto:` fallback with honeypot field (no CAPTCHA, no third-party service).
